# 사진 이름 자동 변경기

폴더 안의 사진을 촬영 시간 순서대로 자동으로 이름 변경해주는 도구입니다.
인터넷 연결 없이 완전 오프라인으로 동작합니다.

## 결과 예시

```
광화문 라이온즈2기-1.jpg
광화문 라이온즈2기-2.jpg
광화문 라이온즈2기-3.jpg
...
```

---

## 다운로드

오른쪽 상단 초록색 `Code` 버튼 → `Download ZIP` 클릭 후 압축 해제

---

## 사용 방법

### 폴더 구조

압축 해제한 파일들을 **본인의 사진 폴더**에 넣어주세요.

```
내 사진 폴더/
├── 광화문 라이온즈2기/
│   ├── 20260507_191206.jpg
│   └── 20260507_191224.jpg
├── KITECH 교육 2회차/
│   ├── 20260507_100201.jpg
│   └── 20260507_100208.jpg
├── 이름변경.bat            ← Windows용
├── rename_photos.ps1       ← Windows용
└── 이름변경_mac.command    ← Mac용
```

---

### Windows

`이름변경.bat`를 **더블클릭**하면 자동으로 처리됩니다.

> 처음 실행 시 Windows 보안 경고가 뜨면 `추가 정보` → `실행` 클릭

---

### Mac

#### 최초 1회 설정 (권한 부여)

터미널을 열고 아래 명령어를 실행하세요.

```bash
chmod +x ~/사진폴더경로/이름변경_mac.command
```

또는 Finder에서 파일을 **우클릭 → 열기** 로 실행하면 보안 경고를 건너뛸 수 있습니다.

#### 이후 사용

`이름변경_mac.command`를 **더블클릭**하면 자동으로 처리됩니다.

---

## 규칙

- **폴더 이름**이 파일명 앞에 붙습니다 → 원하는 이름으로 폴더를 만드세요
- 사진은 **촬영 시간 순서**대로 번호가 매겨집니다
- 지원 형식: `.jpg` `.jpeg` `.png` `.heic` `.gif` `.bmp` `.mp4` `.mov`
