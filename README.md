# 사진 이름 자동 변경기

폴더 안의 사진을 촬영 시간 순서대로 자동으로 이름 변경해주는 도구입니다.

## 결과 예시

```
광화문 라이온즈2기-1.jpg
광화문 라이온즈2기-2.jpg
광화문 라이온즈2기-3.jpg
...
```

## 사용 방법

### 1. 다운로드

오른쪽 상단 초록색 `Code` 버튼 → `Download ZIP` 클릭 후 압축 해제

### 2. 파일 배치

압축 해제한 폴더 안의 파일 2개를 **본인의 사진 폴더**에 넣어주세요.

```
내 사진 폴더/
├── 광화문 라이온즈2기/
│   ├── 20260507_191206.jpg
│   └── 20260507_191224.jpg
├── KITECH 교육 2회차/
│   ├── 20260507_100201.jpg
│   └── 20260507_100208.jpg
├── rename_photos.ps1   ← 여기에 넣기
└── 이름변경.bat        ← 여기에 넣기
```

### 3. 실행

사진을 추가한 뒤 `이름변경.bat`를 **더블클릭**하면 자동으로 처리됩니다.

## 규칙

- **폴더 이름**이 파일명 앞에 붙습니다 → 원하는 이름으로 폴더를 만드세요
- 사진은 **촬영 시간 순서**대로 번호가 매겨집니다
- `이름변경.bat`과 `rename_photos.ps1`은 항상 같은 폴더에 있어야 합니다
- 지원 형식: `.jpg` `.jpeg` `.png` `.heic` `.gif` `.bmp` `.mp4` `.mov`

## 주의사항

- 인터넷 연결 불필요, 완전 오프라인 동작
- Windows 전용 (PowerShell 기반)
- 처음 실행 시 Windows 보안 경고가 뜰 수 있습니다 → `추가 정보` 클릭 후 `실행` 선택
